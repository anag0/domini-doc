# .text()

## Variations

### .text(): ``String``
Gets the textContent value
```javascript
console.log( $( "p" ).text() );
```

### .text( ``String`` v ): ``self``
Sets node textContent to the argument value.
```javascript
$( "p" ).text('Text!');
```
