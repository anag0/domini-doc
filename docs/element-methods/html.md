# .html()

## Variations

### .html(): ``String``
Gets the innerHTML value
```javascript
console.log( $( "p" ).html() );
```

### .html( ``String`` v ): ``self``
Sets node innerHTML to the argument value
```javascript
$( "p" ).html('Link: <a href='#'>Click here!</a>');
```
